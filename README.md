# autotk
AutoTK是使用Android官方API实现自动化处理海外抖音tiktok的app，能实现自动浏览视频、评论、收藏、上传、切换账号等操作。

[tiktok-23.4.4官方下载](https://tiktok.en.uptodown.com/android/download/4487859)
